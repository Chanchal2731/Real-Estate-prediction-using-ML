# Real-Estate-prediction-using-ML

This project focuses on predicting real estate property prices using Machine Learning classification and regression techniques. 
The goal is to assist buyers, sellers, and real estate stakeholders in making data-driven decisions by analyzing historical property data.
This project predicts real estate property prices using Machine Learning.
The dataset includes features like area, location, and number of rooms.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

## Models Used
Linear Regression, Decision Tree, Random Forest

## Conclusion
Machine learning models are used to estimate property prices effectively.
